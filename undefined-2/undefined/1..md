# 📔 1. 본인인증 준비하기

[**`가맹점 식별코드`**](../../undefined/3..md)를 이용하여 `IMP` 객체를 초기화합니다.

{% tabs %}
{% tab title="JavaScript" %}
{% code title="client-side" %}
```javascript
var IMP = window.IMP;      // 생략 가능
IMP.init("{가맹점 식별코드}"); // 예: imp00000000
```
{% endcode %}
{% endtab %}

{% tab title="JS ES Next" %}
{% code title="client-side" %}
```javascript
const IMP = window.IMP;    // 생략 가능
IMP.init("{가맹점 식별코드}"); // 예: imp00000000
```
{% endcode %}
{% endtab %}
{% endtabs %}
